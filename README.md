# A word about enterprises and software development.

Although in many professions, junior hiring is best thought as a balance between cost and performance, in Software development is somehow different. Enterprises has invested a lot of money and time (effort) into current code they have. Throwing away all that code, requires huge investments not every enterprises wishes to do, because it's the opposite to win-win. Customers are sitisfied to what they currently have, it may take : you don't know the details, you probably don't have the code, you will not get any new real benefit 

Doing code from scratch sometimes fits all requirements for small initiatives and it is best to get into new technologies.
